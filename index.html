DNA Duplex Nearest-Neighbor Calculator

Predict thermodynamic parameters for DNA duplexes using selectable buffer or cell-type nearest-neighbor parameter sets. Calculation: sum of NN terms + two terminal/initiation terms + self-complementary correction when applicable.
Results for the selected condition will appear here.
Input rules

Use A/C/G/T or dA/dC/dG/dT. Strand A is always 5′→3′. Default Strand B is written left-to-right as the antiparallel strand directly under A, i.e. 3′→5′. NN lookup uses NN symmetry 5′XY3′/3′ZW5′ = 5′WZ3′/3′YX5′, for example dAG/dTC is matched to dCT/dGA.

Calculation Model

  • ΔH° = Σ(NN ΔH°) + initiation + terminal correction
  • ΔS° = Σ(NN ΔS°) + initiation + terminal correction
  • ΔG°37 = ΔH° − 310.15 × ΔS° / 1000
  • For non-self-complementary duplexes: Tm−1 = [R ln(Ct/4) + ΔS°] / ΔH°
  • For self-complementary duplexes: Tm−1 = [R ln(Ct) + ΔS°] / ΔH°
  • R = 1.987 cal mol−1 K−1; Ct is the total strand concentration.
  • For self-complementary sequences, symmetry correction is applied.
  • Equivalent nearest-neighbor representation is used: 5′XY3′/3′ZW5′ = 5′WZ3′/3′YX5′.
